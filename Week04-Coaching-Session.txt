### Topic 1: Handling Form Submissions

**Learning Coach:**

Please paste a form-handling code snippet.

**Student:**

```javascript
router.post("/project/update/:projectId", async (req, res, next) => {
  try {
    await projectModel.updateProject(req.params.projectId, req.body);
    res.redirect("/projects");
  } catch (err) {
    next(err);
  }
});
```

**Learning Coach:**

What does this code do?

**Student:**

It gets the submitted form data from `req.body`, sends it to the model to update the database, and then redirects to the projects page.
